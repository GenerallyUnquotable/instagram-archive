# Instagram Comments Archive

This project contains a complete, searchable archive of comments from an Instagram post.

## 🔗 View the archive

👉 https://generallyunquotable.github.io/instagram-archive/

## 📦 Contents

* `index.html` – readable version (best for browsing)
* `instagram_comments_deduped_obvious.csv` – structured dataset
* `instagram_comments_possible_duplicates.csv` – review file

## 📌 Notes

* Comments are in chronological order (oldest → newest)
* Replies are nested to reflect conversation threads
* Some timestamps are approximate (based on Instagram “weeks ago” data)
